# Plant Tracker 1.0.1

## Google Play "What's new" (short version)
Plant Tracker is now available on Android. This release adds customizable color themes including pink, improved light mode styling, more flexible watering reminder intervals with years, months, and days, a better household setup flow, Android UI polish, and a refreshed app icon and splash screen.

## Full release notes
Plant Tracker is now available on Android.

What's new in version 1.0.1:
- New color theme support in light mode, so theme choices now carry through more of the app.
- New Pink theme added.
- More flexible watering reminder intervals with years, months, and days.
- Default watering interval settings now apply when creating a new plant.
- Improved household setup flow so turning sharing off resets the setup cleanly.
- Android UI polish for navigation, settings modals, and plant detail layout.
- Refreshed app icon and splash screen.

## Alternate short version
First Android release of Plant Tracker. Customize your app with updated light mode themes and a new Pink theme, set watering reminders with years, months, and days, and enjoy a cleaner household setup flow, improved Android visuals, and refreshed app branding.
